Compare Aura

## Requirements

* Git
* Ubuntu 18.04
* g++-7 (7.5.0 in ubuntu 18.04)
* cmake 3.17
* openssl 1.1.1
* Apache Thrift 0.13.0

## Building

```bash
cd build
# use cmake/make to build the code
cmake ..
make
```
## Usage
We provide a shell command for total keyword search time cost on Crime dataset:

```
sh ./test.sh

```
